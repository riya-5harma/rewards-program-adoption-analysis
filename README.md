# Rewards Program Adoption & Customer Segmentation Analysis

## Overview

This project explores customer adoption patterns for digital reward programs and identifies user segments most likely to participate in loyalty initiatives.

## Business Problem

Digital reward programs often face low adoption rates. Understanding customer behavior helps organizations target the right users and improve engagement.

## Objectives

- Analyze customer behavior
- Identify adoption patterns
- Segment customer populations
- Support campaign targeting
- Improve program participation

- SQL
- Python
- Pandas
- Excel
